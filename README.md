# Proof of Static Indeterminacy Formula Equivalence

This notebook demonstrates the mathematical equivalence of three commonly used formulations for calculating the **Degree of Static Indeterminacy (DSI)** in two-dimensional structural frames.

Using concepts from **structural analysis**, **graph theory**, and **Euler's planar graph formula**, the notebook shows that the following approaches are simply different topological representations of the same structural property:

* Member-based formulation
* Tree-based formulation
* Closed-loop formulation

## What the Notebook Does

* Defines the three static indeterminacy equations used in structural analysis.
* Introduces Euler's relationship for planar structural graphs.
* Derives the connections between members, joints, and closed loops.
* Uses symbolic computation to transform one formulation into another.
* Demonstrates that all three equations produce identical results for any valid planar frame.

## Key Concepts

* Degree of Static Indeterminacy (DSI)

## Purpose

The goal of this notebook is to provide a rigorous mathematical proof that the different static indeterminacy formulas taught in structural analysis courses are equivalent, helping students understand the topological foundations behind these seemingly different approaches.
